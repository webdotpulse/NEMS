# How to get a GitHub Token
To configure NEMS to check for and install updates from a private GitHub repository, you need a GitHub Personal Access Token (classic).
1. Go to your GitHub account settings.
2. Navigate to Developer settings -> Personal access tokens -> Tokens (classic).
3. Generate a new token and give it the `repo` scope to read private repositories.
4. Copy the token and paste it into the NEMS settings UI.
