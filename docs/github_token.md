# How to get a GitHub Token
To configure NEMS to check for and install updates and avoid IP rate limiting from GitHub's API during update checks, you can provide a GitHub Personal Access Token (classic).
1. Go to your GitHub account settings.
2. Navigate to Developer settings -> Personal access tokens -> Tokens (classic).
3. Generate a new token. (You only need the `repo` scope if the target repository is private).
4. Copy the token and paste it into the NEMS settings UI.
